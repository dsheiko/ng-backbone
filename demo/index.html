NG.Backbone / Exoskeleton + SystemJS • Hero Cards

This is a simple app demonstrating NgBackbone. You can enter hero name and power in the form and form gets validated automatically. It requires no additional code in the view. The hero card gets stored in local storage and appears in the list below. You can sort the list, select and remove items. That also almost doesn't need to be supported by view code.
